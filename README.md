# VUE POKER MACHINE AND GAME AUTH HUB

небольшой хаб на 2 игры, с регестрацией по jwt. влючает в себя покерную машину и змейку.

# используемые технологии

- Express
- Sqlite
- Axios
- webpack
- тесты jest

### УСТАНОВКА

установите зависимости и запустите команду npm start

```sh
$ npm install
$ npm start
```
